Implementing a software renderer to understand topics like
rasterization,raytracing and other stuff, goal is to learn 
more about graphics and then move onto graphics APIs

# Run

## Linux
```
./build.sh
```
## Windows
```
.\build.bat
```

# Goo software renderer

## Features
- Draw lines
    - Naive algorithm
- Draw triangles
    - Wireframe
    - Filled
        - Scanline
